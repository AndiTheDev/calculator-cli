
# Calculator CLI

A CLI program to calculate numbers using Python. You can use it by using terminal and it can make easier for you to use it.


## Installation

Install this module first to make it easier to run this program. First, open Powershell on Windows or curl for Linux/Mac. Then, type this command: 

```bash
  pip install uv
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/AndiTheDev/calculator-cli.git
```

Go to the project directory

```bash
  cd calculator-cli
```

Install dependencies

```bash
  uv sync
```

Start the program

```bash
  python main.py
```
